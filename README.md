# Futurama App — The Left Bit Entry Level Test

A simple two-screen React app that displays characters from the Futurama TV series.

## Tech Stack

- React 19
- React Router v6
- SCSS Modules
- PropTypes
- ESLint
- Vite

## Features

- Character list fetched from the Futurama public API
- Character detail page with full info and sayings
- Favourite characters with persistent state via localStorage
- Responsive layout (mobile-first, breakpoint at 600px)

## Getting Started

```bash
npm install
npm run dev
```
